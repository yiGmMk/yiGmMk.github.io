  • 分类

    只有文章支持分类和标签，您可以在 Front-matter 中设置。在其他系统中，分类和标签听起来很接近，但是在 Hexo 中两者有着明显的差别：分类具有顺序性和层次性，也就是说 Foo, Bar 不等于 Bar, Foo；而标签没有顺序和层次，如下，Life是Diary的子类。

    标签插件和 Front-matter 中的标签不同，它们是用于在文章中快速插入特定内容的插件。
